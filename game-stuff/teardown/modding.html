API

Undocumented Functions

IsAchievementCompleted(id)

id - String, achievement ID

if not IsAchievementCompleted("ACH_WATCH_ABOUT") then
	CompleteAchievement("ACH_WATCH_ABOUT")
end

Checks if you already have a specified achievement.

Only works if called by data\script\achievements.lua

CompleteAchievement(id)

id - String, achievement ID

Complete specified achievement.

Only works if called by data\script\achievements.lua

CreateMaterial(material, r1, g1, b1, r2_opt, g2_opt, b2_opt)

material - String, material (e.g. dirt, masonry, etc.)
r1 g1 b1 - RGB value, material color
r2 g2 b2 - RGB value (optional), color used to achieve material (e.g. RBG '20 20 20' produces tarmac)

Used by ground.lua heightmap script to turn the provided image into in-game materials.
